1. What is a function literal?
A function literal is another way to write a function, without the need to define the name of the function. A function literal is created in-line, so you have to call the function after the function declaration. 

2. What is the constructor?
The constructor is a simple way to create objects within the same class/object-type.

3. What does the return statement do?
The return statement in a function is pretty self-explanatory. When you return something in a function, the variable calling the function will get the returned value from the function.

5. Briefly describe scope in JavaScript.
Scope in JavaScript determines which variables is visible in part of the code.
